Ford Design System v1.0.0

Una colección exhaustiva de componentes reutilizables, guías de estilo y estándares de marca diseñados para productos digitales modernos y escalables.

Componentes Modulares

Construido con una arquitectura atómica, permitiendo flexibilidad total desde átomos básicos hasta organismos complejos.

Tokens de Diseño

Gestión centralizada de colores, tipografías y espaciados mediante variables CSS y mapas Sass para una consistencia visual perfecta.

Dark Mode Nativo

Soporte completo para temas claro y oscuro, con transición suave y persistencia de preferencias de usuario.

Documentación Viva

Cada componente está documentado con ejemplos interactivos, snippets de código y guías de uso directas.

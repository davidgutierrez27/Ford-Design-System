Anatomía

Los dropdowns son componentes versátiles que permiten organizar la navegación o selecciones en un menú desplegable.

1

Dropdown Container

  • Height: 34px
  • Width: 210px (Min) / 310px (Max)
  • Border left weight: 2px
  • Border radius: 4px
2

Icon (Home)

  • Style: Icon/Outline/home
  • Visibility: Optional (Boolean)
3

Label

  • Text style: Body/Body 16 Regular
  • Default color: Neutral-500 (#303030)
4

Caret Icon

  • Style: Icon/Outline/caret
  • Behavior: Rotates when menu is open (Points Down by default)

Variantes

Configuraciones de elementos internos según la necesidad de la interfaz.

Completo

Variante estándar con todos los elementos visibles.

Sin Icono

Para menús donde el texto es suficiente identificador.

Sin Caret

Para items que actúan como enlaces directos pero mantienen el estilo de dropdown.

Solo Texto

Máxima simplicidad visual.

Layout y Espaciado

Dropdown Config

  • Direction: Horizontal
  • Alignment: Middle left
  • Vertical resizing: Fixed (34px)
  • Horizontal resizing: Fill
  • Item spacing: 12px
  • Padding: 0 12px

Implementación

HTML Estructural

<!-- Dropdown Completo -->
<div class="dropdown">
  <button class="dropdown__btn dropdown-toggle" data-target="drop1">
    <span class="icon">...</span>
    <span class="label">Menu Item</span>
    <span class="caret">
       <svg ...><g transform="rotate(90 8 8)">...</g></svg>
    </span>
  </button>
  <div class="dropdown__content" id="drop1">
    <!-- Sub-items -->
  </div>
</div>

Accesibilidad

Buenas Prácticas

  • Aria expanded: El JS debe alternar aria-expanded="true/false".
  • Roles: Usar role="button" y role="menu" para el contenido.
  • Teclado: El foco debe mantenerse visible en todo momento.

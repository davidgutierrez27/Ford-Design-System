Icon Actions

  • Gap: 24px (16px on Mobile)

Layout y Espaciado

Puntos de Interrupción (Breakpoints)

  • Desktop (> 768px): Padding 250px, Links visibles.
  • Tablet (480px - 768px): Padding 24px, Burger + Search + User visibles.
  • Mobile (< 480px): Padding 16px, Solo Burger visible.

Propiedades y Estados

La barra de navegación cambia su apariencia al hacer hover para garantizar legibilidad sobre fondos claros.

Estado: Default

  • Bg: Transparent
  • Items: White (#FEFEFE)

Estado: Hover

  • Bg: Neutral-50 (#EAEAEA)
  • Items: Neutral-500 (#303030)

Variantes

Configuraciones adicionales para búsqueda expandida y navegación móvil.

Búsqueda Expandida

  • Clase: .nav-bar--search-active
  • Ancho: Flexible (ocupa espacio central)
  • Input: 44px altura

Navegación Tablet (MD)

  • Interacción: 3 Iconos visibles
  • Breakpoints: 480px - 768px

Navegación Móvil (SM)

  • Interacción: Solo Burger Menu
  • Breakpoints: < 480px

Implementación

HTML Estructural

Uso de flexbox para la distribución de elementos.

<!-- Estructura Base -->
<nav class="nav-bar">
    <button class="nav--icon--button--logo">...</button>
    
    <!-- Búsqueda (Opcional) -->
    <div class="nav-bar__search">
        <div class="nav-bar__search-input-wrapper">
            <input type="text" class="search-field" ...>
        </div>
    </div>

    <div class="nav-bar__links">
        <button class="link--button">Vehículos</button>
        ...
    </div>

    <div class="nav-bar__actions">
        <button class="btn__icon">...</button> <!-- Búsqueda -->
        <button class="btn__icon">...</button> <!-- Perfil -->
        <button class="nav-bar__menu">...</button> <!-- Menú Hamburguesa -->
    </div>
</nav>

<!-- Variante Búsqueda Activa -->
<nav class="nav-bar nav-bar--search-active">
    ...
</nav>

Mejores Prácticas

Dos and Don'ts

  • Do: Mantener un máximo de 5-7 enlaces principales para evitar sobrecarga cognitiva.
  • Do: Utilizar el estado expandido de búsqueda solo cuando sea el objetivo principal de la pantalla.
  • Don't: Cambiar el color de fondo de la nav de forma brusca sin una transición suave.
  • Don't: Ocultar el logo en ninguna de las variantes responsivas.

Accesibilidad

Navegación y Contraste

  • Semántica: Usar la etiqueta <nav> con un aria-label descriptivo.
  • Foco: Asegurar que el estado de foco sea visible tanto en fondo oscuro como claro.
  • Contraste: El cambio a Neutral-500 sobre fondo claro garantiza que se cumplan las pautas WCAG.

Implementación

El Link Button se comporta como un botón pero mantiene el estilo visual de un enlace con estados mejorados.

HTML

<button class="link--button">
    Button Link
</button>

SCSS (Opcional)

Si deseas aplicar los estilos directamente vía mixin o clase padre.

// Los estilos base se encuentran en _link-buttons.scss
.custom-link {
    @extend .link--button;
}

Accesibilidad y Buenas Prácticas

Garantizar que los enlaces sean accesibles es fundamental para una navegación clara por teclado y lectores de pantalla.

Mejores Prácticas

  • Propósito claro: El texto del botón debe indicar claramente el destino o la acción (ej. "Ver catálogo" en lugar de "Más información").
  • Navegación por teclado: Mantener un estado de focus visible (border-bottom 4px) para usuarios que navegan con tabulador.
  • Verbos de acción: Preferiblemente usar verbos que inviten a la interacción.
  • Contraste: Asegurar que el cambio a color Skyview en hover sea lo suficientemente distintivo.

Principios de Radius

Consistencia

El uso sistemático de radios de borde crea una experiencia visual armoniosa y profesional en toda la suite de productos.

Jerarquía Visual

Los radios más grandes suelen aplicarse a contenedores principales, mientras que los pequeños se reservan para elementos internos.

Tactilidad Digital

Las esquinas redondeadas suavizan la interfaz, haciéndola sentir más moderna, accesible y amigable para el usuario.

Escalabilidad

Nuestra escala está diseñada para funcionar desde micro-interacciones hasta grandes layouts estructurales.

Roles de Uso

Cada token de radio tiene un propósito específico dentro del sistema para asegurar que la estética se mantenga coherente.

Small Radius (2px - 4px)

Utilizado para detalles pequeños y elementos de interfaz compactos.

  • Checkboxes y Radio buttons
  • Inputs de formularios
  • Tooltips

Medium Radius (8px - 14px)

El estándar para la mayoría de los componentes interactivos y contenedores pequeños.

  • Botones principales y secundarios
  • Cards de contenido pequeño
  • Dropdowns y Menús flotantes

Large Radius (16px - 32px)

Reservado para grandes contenedores y superficies que estructuran la página.

  • Modales y Diálogos
  • Cards principales de dashboard
  • Secciones destacadas con fondo

Implementación

Al igual que otros tokens, el radio debe consumirse preferiblemente a través de variables CSS para permitir flexibilidad.

Tokens de Radius

Lista de variables disponibles para aplicar en SCSS o CSS inline.

Token Valor Uso común
var(--radius-4) 4px Pequeños componentes / Inputs
var(--radius-12) 12px Cards / Botones estándar
var(--radius-24) 24px Grandes superficies / Modales

Ejemplo SCSS

Cómo aplicar el radio en tus estilos.

.my-card {
    background: var(--color-background-white);
    border-radius: var(--radius-12);
    padding: var(--space-24px);
    box-shadow: var(--shadow-card);
}

.my-button {
    border-radius: var(--radius-8);
    // ... otros estilos
}

Buenas Prácticas

✓ Qué hacer
  • Usa radios consistentes dentro de un mismo componente.
  • Asegura que el radio interior sea menor que el exterior si hay anidamiento.
  • Mantén la coherencia con el estilo de marca (Ford usa radios suaves pero definidos).
✗ Qué evitar
  • No mezcles estilos de bordes (rectos y redondos) sin un propósito de jerarquía.
  • Evita radios excesivamente grandes que rompan la legibilidad del contenido.
  • No ignores la escala: usa siempre los tokens definidos.
